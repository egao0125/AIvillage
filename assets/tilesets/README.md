# Tilesets

Pixel art tile sheets for the map. 16x16 or 32x32 tiles.

## Contributing tilesets

- Use [Aseprite](https://www.aseprite.org/) or similar pixel art tool
- Export as PNG sprite sheet
- Include the source `.aseprite` or `.psd` file
- Tile size: 32x32 preferred
- Style reference: Stardew Valley, Habitica

If using open-source tilesets, credit the original author here.
