# Sprites

Character sprite sheets for agents.

## Required animations per character

- Walk: up, down, left, right (4 frames each)
- Idle: front-facing (2-4 frames)
- Sit: for cafés and benches
- Talk: mouth movement or gesture
- Sleep: lying down

## Spec

- Size: 32x48 per frame (or 32x32 for chibi style)
- Format: PNG sprite sheet
- Include source file (.aseprite)
